# House Price Prediction using Machine Learning

## Objective
Predict house prices using machine learning techniques.

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Visualizations
- Histogram of house prices
- Correlation heatmap
- Actual vs Predicted scatter plot
